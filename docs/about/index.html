CRMint is a platform to make the integration of your customer and marketing data easy. It runs on top of Google Cloud and allows you to develop and run pipelines on this data, for eventual output into a range of products, including Google Analytics, Google Ads, and Campaign Manager.

The main usage scenarios:

  • Incorporating your customer data from your CRM to generate remarketing strategies

  • Automating the Offline-to-online conversion process

  • < .... something ML based ... >

Core Features that make it awesome:

  • Easy to deploy on Google Cloud

  • Completely extensible to meet your custom requirements

  • A range of existing templates for common use cases

Technical background

In an essence, CRMint is a Google App Engine application written in Python for Google App Engine Python Standard Environment. CRMint App uses Cloud SQL to store it's business data, and Stackdriver Logging to store it's activity logs.
